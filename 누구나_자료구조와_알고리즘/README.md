## 누구나 자료구조와 알고리즘

| 챕터 | 내용                                         |
| ---- | -------------------------------------------- |
| 1장  | 자료 구조가 중요한 까닭                      |
| 2장  | 알고리즘이 중요한 까닭                       |
| 3장  | 빅 오 표기법                                 |
| 4장  | 빅 오로 코드 속도 올리기                     |

| 8장  | 스택과 큐로 간결한 코드 생성                 |

| 10장 | 속도를 높이는 재귀 알고리즘                  |
| 11장 | 노드 기반 자료 구조                          |
